Defensive Detection Rules for Splunk Enterprise Security

Overview

This app contains detection rules automatically converted from Sigma format and organized by MITRE ATT&CK framework. The rules are designed to integrate seamlessly with Splunk Enterprise Security.

Features

  • MITRE ATT&CK Integration: Rules are tagged with techniques and tactics
  • Notable Events: Generates ES notable events for security incidents
  • Correlation Rules: Advanced correlation for multi-stage attacks
  • Customizable Macros: Easy adaptation to your environment

Installation

$SPLUNK_HOME/bin/splunk install app defensive-rules-es.tar.gz
$SPLUNK_HOME/bin/splunk restart

Configuration

Important: Review and customize the search macros in macros.conf to match your index names and log sources before enabling the detection rules.

Common customizations:

  • Update index names in macros.conf
  • Adjust search schedules based on your environment
  • Fine-tune alert thresholds to reduce false positives
  • Configure notification actions for high-priority alerts

MITRE ATT&CK Coverage

This app provides detection coverage for multiple MITRE ATT&CK techniques across various tactics. Check the generated mitre-coverage.json file for detailed technique coverage.
